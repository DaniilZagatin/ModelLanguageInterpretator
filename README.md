# Model Language Interpretator

Учебный проект по реализации лексического анализатора для простого модельного языка программирования на C++.

Проект был выполнен в рамках изучения основ построения интерпретаторов и компиляторов. Основная цель - разобрать исходный текст программы на последовательность лексем: ключевые слова, идентификаторы, числа, строки, операторы и разделители.

## Что реализовано

- лексический анализатор на C++;
- разбор исходного файла с программой;
- распознавание ключевых слов модельного языка;
- распознавание идентификаторов;
- поддержка целых и вещественных чисел;
- поддержка строковых литералов;
- поддержка комментариев вида `/* ... */`;
- распознавание операторов и разделителей;
- базовая обработка лексических ошибок;
- проверка баланса фигурных скобок.

## Поддерживаемые типы

В модельном языке предусмотрены следующие типы:

- `int`
- `real`
- `string`

Отдельный тип `bool` не используется: значения типа `int` интерпретируются как логические. Числа больше `0` считаются истиной, значения `0` и меньше — ложью.

## Пример программы

```txt
program {
    int x = 1;
    real y = 1.11;
    string s = "abc";

    while (x > 0) {
        y = y + 1;
    }
}
```
